Introduction

オルテンシア学園とは

（チーム＝学園）（メンバー＝生徒）（マネージャー＝生徒会委員）（チームマスター＝生徒会長）として、 ＰＳＯ２の世界で、学園生活を楽しもうかなぁー、てなところが「オルテンシア学園」です。プレイヤーの個性を尊重し、プレイヤーをＰＳＯ２世界の、１人のキャラクターとします。それぞれのプレイヤーが、それぞれのプレイスタイルで楽しく遊べ、集える場所とします。一般生徒（コモンメンバー）以下には、一切の強制はしませんが、プレイヤーとしてのマナーは、もちましょう。

※ チームイベントの開催、チームトロフィーの取得など、今は、（2013.8/15現在）考えてません。
